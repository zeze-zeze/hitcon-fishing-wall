釣魚牆介紹

本活動純屬教育性質，目的在於期望提升 HITCON 與會者對 WiFi 安全性的資安意識。我們將把釣魚 WiFi 的會眾輸入的帳號放到 dashboard 上，並於 HITCON CMT 2023 結束後將資料刪除不留備份。


資安宣導

1. 不使用來路不明的 WiFi。
2. 檢查網址與憑證是否正確以辨認釣魚網站。
3. 開啟二階段驗證 (2FA) 以增加帳號安全性。


官方 WiFi

請記得連線時使用 HITCON 官方提供的 WiFi，別再不小心走進釣魚牆囉！

SSID: HITCON 2023

更改 Description

輸入下方欄位更改 description，可以更改使用者的敘述，簡短填寫任何你想說的話。更改的使用者敘述會是你最後登入釣魚網站的使用者名稱。


提交 Flag

最後為了證明會眾的密碼並沒有被蒐集，我們把 flag 藏在各釣魚網站中，找到它們後提交於下方欄位。提交 flag 的使用者會是你最後登入釣魚網站的使用者名稱。
