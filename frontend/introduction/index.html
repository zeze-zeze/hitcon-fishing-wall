介紹

本活動純屬教育性質，目的在於期望提升 HITCON 與會者對 WiFi 安全性的資安意識。誤入釣魚 WiFi 的參與者們，我們將把你們的帳號去識別化後放到 dashboard 上，並於 HITCON CMT 2023 結束後將資料刪除不留備份。

活動方式

釣魚牆偽造 HITCON 官方 WiFi 以及一些登入頁面，前端收集登入驗證資料，後端負責接收並處理這些資料，同時進行帳戶的去識別化，並將其顯示在 dashboard 上。
釣魚牆的 dashboard 一共有三個欄位，時間 (time)、使用者 (username)、敘述 (description)，可以指定任意使用者並更改其敘述，方法如下： curl -X POST http:// : /fish -d "username= &description= "

宣導事項

WiFi 能夠讀取封包資訊，例如瀏覽網站以及輸入內容，請在連線前辨別 WiFi 的名稱是否與官方提供的名稱一致，並在登入網頁後查看該網站的憑證是否正確，以此辨別是否使用到不安全的連線。

結語

希望能讓大家更加了解釣魚牆的運作方式，並提高資訊安全意識。若有任何進一步的問題或疑慮，請隨時向工作人員詢問。
